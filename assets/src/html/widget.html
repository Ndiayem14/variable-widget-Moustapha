What is a Variable?

A variable is like a labeled storage box. You can put information inside it, check what's in it, or change what's stored anytime you want!

The box has a name (so you know which box it is), and it holds a value (what's inside the box).

The cool part? The name stays the same, but you can change the value whenever you want!
